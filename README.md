## ⏳ Overview

GPUPS is the benchmark application for [OpenAI Triton](https://github.com/openai/triton). The GPUPS is a portmanteau of
GPU and FLOPS. Check [Wiki](https://github.com/daemyung/gpups/wiki) for performance results.

## 💻 Requirements

- Platform: Linux
- Hardware: NVIDIA
